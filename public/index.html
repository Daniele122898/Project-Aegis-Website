What is Project-Aegis?

Warning System

Project-Aegis is supposed to act as a warning system. Users and Bot Creators can report Guilds using the Guild ID either via the Dashboard or the API.

A report must always contain a good reason like text logs or links to screenshots, anything that can provide reasonable proof to the claims. Every report is logged to the respective user thus if a user makes too many false reports, he gets locked. The better and numerous the reports are the better the API gets. In the first couple months the API is moderated, and Security Levels are only given by Admins. Later, once there is enough data, the system will be automated.

If a Bot joins a Guild he can make an API call and check the security level if there is one set or make an educated guess with the number of open reports and with that decide on whether to leave the Guild or stay connected. With the WebSocket integration the Bot also gets notified on every Guild Security Level update that is performed and can then check if he is part of said Guild.

Security Levels

0 - Good:Starting level. No action needed
1 - Suspicious:Some record of spamming. Not harmful yet.
2 - Moderate:Some spamming and misconduct.
3 - High:We advice to leave these Guilds.
4 - Extreme:We strongly advice to leave these Guilds.
